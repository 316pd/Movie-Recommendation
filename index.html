About Data

This is a TMDB 5000 Movie Dataset taken from kaggle, dataset contains information of cast, crew, summary etc.

Contents

The file tmdb_5000_movies.csv contains ~5000 rows and 20 columns. Each row corresponds to an individual movie. The columns are:

  1. budget - budget of the movie
  2. genres - genres of the movie
  3. homepage - official homepage of the movie if exist else NaN
  4. id - id of the movie A/Q to TMDB website
  5. keywords - movie description keywords
  6. original_language - language
  7. original_title - name of the movie
  8. overview - short summary of movie
  9. popularity - Team name
  10. production_companies - production companies
  11. production_countries - production countries
  12. release_date - release date
  13. revenue - revenue
  14. runtime - runtime
  15. spoken_languages - spoken language
  16. status - Released or Not
  17. tagline - short tags
  18. title - name of movie
  19. vote_average - voting scale 0 to 10
  20. vote_count - voting count of movies

The file tmdb_5000_credits.csv contains ~5000 rows and 4 columns. The columns are:

  1. movie_id - budget of the movie
  2. title - genres of the movie
  3. cast - all characters of movie
  4. crew - director etc of movie
